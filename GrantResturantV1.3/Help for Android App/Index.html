    • 12. Push Notification Configuration top


    • step 1: Goto Google APIs Console page and create a new project. (If you have already created other project you will see below page, otherwise it will take you to dashboard)

    • step 2: After creating project you can see the project id. (After # Value)

      Note down the project id and add in Android Project as below.

    • Open CommonUtilities class under Utils package name.
    • Add Project id in SENDER_ID.

    • step 3: turn on Google Cloud Messaging api for Android.

    • step 4: Now go to Credentials >> create credentials >> select API key
    • step 5: Now you will see a pop up window named Create a new key. Select Server Key option out of four option.
    • step 6: Create server api key page give suitable name as your choice and second field is server ip address(which is optional).After Go with Create button.
    • step 7: Lastly you will get a pop up window with generated server api key.
    • step 8: Once you are done, click on API Access and note down the API Key. This API key will be used when sending requests to GCM server.

    • step 9:This api key put in admin panel notification field
