A) Firebase Database - top

1- Create a free Firebase account

2- Create new project

3- Go to Database tab and import json file from "/upload/db.json"

4- Go to RULES tab change it to

{ "rules": { ".read": true, ".write": true, } }

B) Ionic config - top

1- Get firebase config

a- Go to Project settings

b-Go to Add firebase to your web app

- Copie (apiKey,authDomain,databaseURL,storageBucket)

2- Open MyFoodDevlivery/www/index.html

3- Change (apiKey,authDomain,databaseURL,storageBucket) Line 43

4- Open MyFoodDevlivery/www/js/service.js and MyFoodDevlivery/www/js/controller.js Line 172

3- Change

<your-firebase-app> by your Firbabase databaseURL

5- Change App name and widget id in MyFoodDevlivery/config.xml

6- Create new app here and get the ID

7- Change the ID and app name in MyFoodDevlivery/ionic.project

C) Backend - top

1- Change

<your-firebase-app> by your Firbabase databaseURL in upload/backend/app/dashboard/dashboard.js and upload/backend/app/dashboard/dashboard_category.js in Line (6,7,14,84,94,154,203,236,238,270,278,306,308)

2- Upload the Upload folder to your hosting

3- Default email and password is email: admin@hamrounihani.com password: codecanyon

4- Change the login and password in backend/app/data/user.php

5- Is very easy to to customize your app add,edit catergory recipe and users
