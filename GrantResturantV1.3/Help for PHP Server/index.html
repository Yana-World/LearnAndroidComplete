A) Installing MAMP/WAMP - top

There are a lot of available installation online and it is a easy as downloading the installer and installing it

After the installation like in MACOSX, the MAMP appears like this.

B) Creating a Database - top

Create a database and named as "db_restaurant"

Note :Created database name as sql file name.

C) Importing the SQL File - top

The file SQL File found at the Redixbit IOS /MySQL SQL File/

Click the db_restaurant then select Import

Select the db_restaurant.sql then click Open

Press the Go Button then wait for confirmation

IF all is working good then you will see all the tables in the Structure Tab

D) Changing the Config File - top

You can locate the config file at this folder Redixbit iOS /PHP Scripts/grant restaurant/application/config.php

Open the file in your favorite editor then change the content of it

Modify the content inside the red boxes depending on your settings of your server or your localhost

E) Live Server Creating Database - top

Click to open mysql database wizard to open form fill out your database detail following this example.

Enter Redixbit ios / Mysql Sql File Name As Database Name. Ex:db_Restaurant

F) Live Server Creating Database User - top

creating a database user check this example.

Note: any database username are allowed but databse connection setting username field same username as you want to create database user.

G) Live Server add database user - top

Add user into your database.select your database name and database and click to add user button and go to manage user privileges box.

select all privileges and press click to make changes button.show massage for database user add successfully.

G) Live Server Database Connection Setting - top

You can locate the config file at this folder Redixbit iOS /PHP Scripts/grant restaurant/application/config.php

you can write hostname ,username , password and database name as you can create.
